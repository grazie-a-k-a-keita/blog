## 入力方法

日本語入力の状態で、`z` + `l`と入力すると「→」に変換できる。

他の矢印も同じ感じでできるらしい。

| key       | 記号 |
| --------- | ---- |
| `z` + `h` | ←    |
| `z` + `k` | ↑    |
| `z` + `j` | ↓    |

Vimのカーソル移動と同じらしい（Vimは使わないから知らなかった...）
